# WeatherMind AI 🌤️

An AI-powered global weather application with real-time forecasts 
and a natural language AI assistant powered by Claude.

## Features
- 📍 Auto-detect location + search any city worldwide
- 🌡️ Current conditions — temperature, humidity, wind, UV index, pressure
- ⏱️ 24-hour hourly forecast
- 📅 5-day forecast
- 🤖 Claude AI assistant — ask weather questions in plain English

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Python + FastAPI
- Weather Data: OpenWeatherMap API
- AI: Anthropic Claude API
- Deployed: GitHub Pages + Render
